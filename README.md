# 🏰 LuxeEstate AI — Premium Real Estate Investment Engine

LuxeEstate AI is a state-of-the-art, AI-orchestrated investment platform designed for the ultra-high-net-worth individual. It transforms property discovery into a precision-engineered investment journey by blending cutting-edge aesthetics with deep-learning analytics.

![Dashboard Preview](https://images.unsplash.com/photo-1613490493576-7fde63acd811?auto=format&fit=crop&q=80&w=1600)

## 💎 Project Philosophy
LuxeEstate AI bridges the gap between emotional desire and financial logic. Through a "Platinum" user interface and real-time AI insights, we provide investors with a data-driven view of global luxury assets.

## 🚀 Key Features

- **🧠 AI Neural Search**: Contextual search engine that understands natural language (e.g., *"Find a villa with sunset views and a private gym"*).
- **📈 ROI Predictor Analytics**: Real-time 5-year valuation curves, monthly rent estimates, and break-even timelines.
- **🍱 Bento Grid UI**: A modernist, glassmorphism-based layout that prioritizes high-fidelity property imagery.
- **🏦 Multi-Currency Global Engine**: Seamlessly toggle between USD ($), AED (AED), and INR (₹) across the entire platform.
- **👤 Agent Dashboard (Active Listings)**: Professional workspace for managing leads, paginate listings, and track market performance.
- **🌿 AI Lifestyle Insights**: Environmental scoring for Air Quality Index (AQI), Serenity (Noise), and Greenery Density.
- **💬 AI Concierge**: Persistent property advisor contextually aware of every listing in the database.

## 🛠️ Technical Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion, Lucide React, Recharts.
- **Backend**: Node.js, Express, MongoDB Modern (Mongoose).
- **Authentication**: JWT-based secure auth with Role-Based Access Control (RBAC).

## 📦 Installation & Setup

### 1. Clone the Asset
```bash
git clone https://github.com/your-username/luxeestate-ai.git
cd luxeestate-ai
```

### 2. Backend Configuration
```bash
cd backend
npm install
# Create a .env file based on .env.example
# PORT=5000
# MONGO_URI=your_mongodb_atlas_uri
# JWT_SECRET=your_super_secret_key
npm start
```

### 3. Frontend Configuration
```bash
cd ../frontend
npm install
# Create a .env file based on .env.example
# VITE_API_URL=http://localhost:5000
npm run dev
```
